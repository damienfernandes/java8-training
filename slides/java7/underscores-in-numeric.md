### Numeric literals

---

### Before Java 7

```java
int tenMillion = 10000000;
System.out.println("Amount is " + tenMillion);
```

```bash
> Amount is 10000000
```

---

### With Java 7

```java
int tenMillion = 10_000_000;
System.out.println("Amount is " + tenMillion);
```

```bash
> Amount is 10000000
```
